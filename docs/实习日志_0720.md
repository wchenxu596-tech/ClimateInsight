# 实习日志

**日期**：2026年7月20日

今天完成了 ClimateInsight 全球气候智能分析平台的核心开发工作。项目基于 NOAA GSOD 全球气候观测数据集，覆盖2024年全年12,160个气象站的1亿条原始记录，构建了 Hive 数据仓库四层架构（ODS→DWD→DWS→ADS），实现数据清洗、单位转换、质量标记与气象标记解析。为解决学生机 YARN 内存限制，采用 Python 流式聚合方案将360万条明细数据汇总为13.8万条月度统计并同步至 MySQL。

后端使用 Flask 蓝图架构重构，实现健康检查、KPI 看板、月度趋势、站点排名、气候带分布和受控 AI 分析助手共7个 API 端点，全部采用参数化查询和年份过滤。前端基于 Vue3 + ECharts 5 + Element Plus 构建五页可视化大屏，包含加载骨架、错误重试和空数据状态处理，并将所有站点名称和气候带分类翻译为中文。Agent 模块采用规则优先的意图识别策略，通过工具白名单确保安全性，输入"DELETE"等危险操作可被正确拒绝。项目已通过 test client 全链路验证并推送至 GitHub 团队仓库，明天将进行前后端联调和答辩准备。
