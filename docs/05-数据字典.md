# ClimateInsight 数据字典

> 数据来源: NOAA GSOD 2015-2025 (11年)  
> 数据仓库: MySQL 8.0 — Python ETL 直写 DWS/ADS (无 Hive ODS/DWD 中间层)

---

## 数据流水线

```
NOAA GSOD tar.gz (每年约 500MB)
  │
  ▼ Python ETL (scripts/download_and_load.py)
  │   · 下载 → 解压 → 清洗(F°→°C, in→mm) → 聚合
  │   · 气候带分类 → 逐月汇总 → 写入 MySQL
  │
  ├── MySQL DWS 层 · dws_station_monthly (站×月汇总)
  │
  └── MySQL ADS 层 (ETL 写入时一并生成)
      · ads_kpi / ads_monthly_trend / ads_ranking
      · ads_zones / ads_zone_trends / ads_stations
```

---

## DWS 层 (MySQL)

### dws_station_monthly — 气象站月度汇总

| 字段 | 类型 | 说明 |
|------|------|------|
| station_id | VARCHAR(20) | 气象站ID (USAF-WBAN) |
| station_name | VARCHAR(200) | 气象站名称 |
| year | INT | 年份 |
| obs_month | INT | 月份 (1-12) |
| latitude | DOUBLE | 纬度 |
| longitude | DOUBLE | 经度 |
| avg_temp | DOUBLE | 月均温 (°C) |
| avg_temp_max | DOUBLE | 月均最高温 (°C) |
| avg_temp_min | DOUBLE | 月均最低温 (°C) |
| max_temp | DOUBLE | 月最高温 (°C) |
| min_temp | DOUBLE | 月最低温 (°C) |
| total_precip | DOUBLE | 月总降水量 (mm) |
| rainy_days | INT | 降雨天数 |
| extreme_days | INT | 极端天气天数 (冰雹+雷暴) |
| heat_wave_days | INT | 热浪天数 (日最高温≥35°C) |
| cold_wave_days | INT | 寒潮天数 (日最低温≤-10°C) |
| frost_days | INT | 霜冻天数 |
| snow_days | INT | 降雪天数 |
| thunder_days | INT | 雷暴天数 |
| obs_days | INT | 实际观测天数 |
| climate_zone | VARCHAR(20) | 气候带分类 |

主键: `(station_id, year, obs_month)`

**气候带分类规则 (按纬度简化):**

| 值 | 中文 | 判定条件 |
|----|------|----------|
| tropical | 热带 | \|纬度\| < 23.5° |
| temperate | 温带 | 35° ≤ \|纬度\| < 50° |
| arid | 干旱 | 23.5° ≤ \|纬度\| < 35° |
| continental | 大陆性 | 50° ≤ \|纬度\| < 66.5° |
| polar | 寒带 | \|纬度\| ≥ 66.5° |

## ADS 层 (MySQL)

### ads_kpi — KPI 看板

| 字段 | 类型 | 说明 |
|------|------|------|
| kpi_name | VARCHAR(50) | 指标名称 |
| kpi_value | DOUBLE | 指标值 |
| kpi_unit | VARCHAR(20) | 单位 |
| kpi_desc | VARCHAR(200) | 中文描述 |
| data_year | INT | 数据年份 |

主键: `(kpi_name, data_year)`

**4 项年度 KPI (每年一条):**

| kpi_name | 说明 | 单位 |
|----------|------|------|
| global_avg_temp | 全球年平均温度 | °C |
| total_stations | 活跃气象站总数 | 个 |
| extreme_event_pct | 极端天气日数占比 | % |
| hottest_station_temp | 年度最高气温 (单站全年最高) | °C |

---

### ads_ranking — 站点排名

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 自增主键 |
| category | VARCHAR(20) | 排名类别 |
| rank_num | INT | 排名序号 (1-15) |
| station_id | VARCHAR(20) | 气象站ID |
| station_name | VARCHAR(200) | 气象站名称 |
| value | DOUBLE | 排名指标值 |
| data_year | INT | 数据年份 |

唯一键: `(category, data_year, rank_num)`  
索引: `idx_cat_year (category, data_year)`

**4 类排名 (每类 TOP15):**

| category | 说明 | 排序规则 | 单位 |
|----------|------|----------|------|
| hottest | 最热站点 | 年均最高温降序 | °C |
| coldest | 最冷站点 | 年均最低温升序 | °C |
| rainiest | 降水最多 | 年总降水量降序 | mm |
| most_extreme | 极端天气最多 | 年极端天数降序 | 天 |

---

### ads_monthly_trend — 月度温度趋势

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| obs_month | INT | 月份 (1-12) |
| avg_temp | DOUBLE | 全球月均温 (°C) |
| avg_max | DOUBLE | 全球月均最高温 (°C) |
| avg_min | DOUBLE | 全球月均最低温 (°C) |

主键: `(data_year, obs_month)`

---

### ads_zones — 气候带分布

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| climate_zone | VARCHAR(20) | 气候带 |
| station_count | INT | 该气候带站点数量 |

主键: `(data_year, climate_zone)`

---

### ads_zone_trends — 气候带趋势 (预聚合)

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| climate_zone | VARCHAR(20) | 气候带 |
| avg_temp | DOUBLE | 该气候带年均温 (°C) |
| avg_precip | DOUBLE | 该气候带年均降水量 (mm) |
| extreme_days | INT | 极端天气总天数 |
| heat_wave_days | INT | 热浪总天数 |
| cold_wave_days | INT | 寒潮总天数 |
| station_count | INT | 该气候带站点数 |

主键: `(data_year, climate_zone)`

说明：从 `dws_station_monthly` 按气候带聚合，用于加速趋势/气候带页面的多年查询。每个气候带每年一条记录。

---

### ads_stations — 站点概要 (预聚合)

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| station_id | VARCHAR(20) | 气象站ID |
| station_name | VARCHAR(200) | 气象站名称 |
| lat | DOUBLE | 纬度 |
| lon | DOUBLE | 经度 |
| climate_zone | VARCHAR(20) | 气候带 |
| avg_temp | DOUBLE | 年均温 (°C) |
| total_precip | DOUBLE | 年总降水量 (mm) |
| risk_events | INT | 风险事件总数 (热浪+寒潮+极端) |
| heat_wave_days | INT | 热浪天数 |
| cold_wave_days | INT | 寒潮天数 |
| extreme_days | INT | 极端天气天数 |
| frost_days | INT | 霜冻天数 |
| snow_days | INT | 降雪天数 |
| thunder_days | INT | 雷暴天数 |

主键: `(data_year, station_id)`  
索引: `idx_year (data_year)`, `idx_zone (data_year, climate_zone)`

说明：从 `dws_station_monthly` 按站聚合的年度概要，用于地图散点渲染和预警风险计算。每站每年一条记录。

## 索引明细

### dws_station_monthly 索引

| 索引名 | 列 | 用途 |
|--------|----|------|
| PRIMARY | (station_id, year, obs_month) | 站×年×月唯一标识 |
| idx_year_month | (year, obs_month) | 按年月范围查询 |
| idx_climate_zone | (year, climate_zone) | 气候带聚合查询 |
| idx_temp_max | (year, avg_temp_max) | 最高温排序 / 排名计算 |
| idx_precip | (year, total_precip) | 降水量排序 / 排名计算 |
| idx_year_sid | (year, station_id) | 站级 GROUP BY 加速 |
