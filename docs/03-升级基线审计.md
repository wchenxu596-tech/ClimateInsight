## 5. 调用链记录

```
用户提问 → AIPanel.vue (frontend)
  → POST /api/agent/query {question, year, page}
  → routes/agent.py: 参数校验 + SQL注入防护
  → agent/service.py: query()
    → agent/intents.py: detect() — LLM优先(DeepSeek, 5s超时) → 规则回退
    → agent/tools.py: 8个工具，参数化SQL
    → agent/responder.py: make_response() — 文字+表格+图表
  → 返回 {answer, intent, table?, chart?}
  → AIPanel.vue: 渲染文字 + el-table + v-chart
```

---

## 6. 安全边界记录

- SQL: 100% 参数化查询 (`query_dict("SELECT ... WHERE x=%s", (val,))`)
- 输入: 500 字符限制, SQL 关键字黑名单
- 意图: JSON Schema 校验 (VALID_INTENTS + EXPECTED_KEYS)
- 年份: 白名单 {2015..2025}
- LLM: temperature=0, max_tokens=250, 输出不直接执行
- 后端缓存: @cached(300) 内存缓存
- 前端缓存: cachedGet() 30s + 请求去重

## 7. 已知限制

| 限制 | 说明 |
|---|---|
| 数据源 | 仅 GSOD，无 GHCN-Daily |
| 粒度 | 仅月粒度（DWS），无日粒度查询 |
| 空间过滤 | 仅气候带/洲，无 bbox/行政区 |
| 对话 | 单轮，无上下文记忆 |
| 报告 | 无异步导出 (Celery) |
| 知识库 | 无 RAG (Milvus) |
| 认证 | 无用户/租户隔离 |
