# ClimateInsight 部署运行手册

> 最后更新: 2026-07-21  
> 项目名称: ClimateInsight — 基于 NOAA GSOD 数据的全球气候智能分析平台  
> 数据范围: 2024 年

---

## 环境要求

| 组件 | 版本要求 |
|------|----------|
| Docker | 20.10+ |
| Docker Compose | 2.0+ |
| Python | 3.11+ (本地ETL) |
| Node.js | 20+ (前端构建) |

---

## 快速启动 (Docker)

```bash
# 1. 克隆仓库
git clone <仓库地址>
cd bi_hub

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env，修改 MYSQL_PASSWORD 和 MYSQL_ROOT_PASSWORD

# 3. 启动全部服务
docker compose up -d --build

# 4. 验证服务状态
docker compose ps
curl http://localhost:8080/api/health

# 5. 浏览器访问
# http://localhost:8080
```

---

## 本地开发启动

```bash
# 1. 确保 MySQL 运行且已建表
mysql -u root -p < sql/mysql/001_init.sql

# 2. 运行 ETL (需Hive有数据)
$env:MYSQL_PASSWORD='your_password'
python scripts/etl_to_mysql.py --year 2024

# 3. 数据校验
python scripts/verify_data.py --year 2024

# 4. 启动后端
cd backend
$env:MYSQL_PASSWORD='your_password'
pip install -r requirements.txt
python app.py

# 5. 启动前端 (新终端)
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

---

## 生产构建

```bash
# 前端构建
cd frontend
npm ci
npm run build
# 产物在 dist/ 目录

# 后端直接使用 gunicorn
cd backend
gunicorn --bind 0.0.0.0:5000 --workers 4 app:app
```

## Docker 服务说明

| 服务 | 容器名 | 端口 | 说明 |
|------|--------|------|------|
| mysql | climate_mysql | 3307→3306 | MySQL 8.0, 命名卷持久化 |
| backend | climate_backend | 5000 | Flask + gunicorn |
| frontend | climate_frontend | 8080→80 | Nginx + Vue3 SPA |

python scripts/etl_to_mysql.py --year 2024

python scripts/verify_data.py --year 2024

## 常用命令

```bash
# 查看日志
docker compose logs -f backend
docker compose logs -f frontend

# 重启服务
docker compose restart backend

# 进入 MySQL
docker exec -it climate_mysql mysql -u root -p climate_dw

# 停止所有服务
docker compose down

# 删除数据卷(重置数据库)
docker compose down -v
```

## 故障排查

**Q: 前端页面加载但数据为空**  
A: 检查后端是否启动 → `curl http://localhost:5000/api/health`

**Q: ETL 报 "Unknown column"**  
A: 执行增量升级脚本 `mysql -u root -p climate_dw < sql/mysql/002_add_dws_weather_days.sql`

**Q: Docker MySQL 端口冲突**  
A: 修改 `docker-compose.yml` 中 MySQL 端口映射 (`3307:3306` 改为其他端口)
