# ClimateInsight 数据字典

> 数据来源: NOAA GSOD 2024  
> 数据仓库: Hive ODS/DWD → MySQL DWS/ADS

---

## ODS 层 (Hive)

### ods_climate_raw — 原始观测数据

| 字段 | 类型 | 说明 |
|------|------|------|
| station_id | STRING | 气象站ID (USAF-WBAN) |
| obs_date | STRING | 观测日期 yyyy-MM-dd |
| latitude | STRING | 纬度 |
| longitude | STRING | 经度 |
| elevation | STRING | 海拔(米) |
| station_name | STRING | 气象站名称 |
| temp_mean | STRING | 日平均温度(°F) |
| temp_max | STRING | 日最高温度(°F) |
| temp_min | STRING | 日最低温度(°F) |
| precip | STRING | 降水量(英寸) |
| weather_flags | STRING | 气象标记(FRSHTT) |

分区键: `year`

---

## DWD 层 (Hive)

### dwd_climate_daily — 清洗后明细数据

| 字段 | 类型 | 说明 |
|------|------|------|
| station_id | STRING | 气象站ID |
| station_name | STRING | 气象站名称 |
| obs_date | STRING | 观测日期 |
| latitude | DOUBLE | 纬度 |
| longitude | DOUBLE | 经度 |
| temp_mean | DOUBLE | 日均温(°C) |
| temp_max | DOUBLE | 日最高温(°C) |
| temp_min | DOUBLE | 日最低温(°C) |
| precip | DOUBLE | 降水量(mm) |
| is_rainy | INT | 是否降雨(0/1) |
| is_extreme | INT | 是否极端天气(0/1) |
| is_heat_wave | INT | 是否热浪(≥35°C) |
| is_cold_wave | INT | 是否寒潮(≤-10°C) |
| has_frost | INT | 是否有霜 |
| has_snow | INT | 是否有雪 |
| has_thunder | INT | 是否有雷暴 |
| obs_month | INT | 观测月份 |
| season | STRING | 季节 |
| quality | STRING | 数据质量(valid/suspicious) |

分区键: `year`

## DWS 层 (MySQL)

### dws_station_monthly — 气象站月度汇总

| 字段 | 类型 | 说明 |
|------|------|------|
| station_id | VARCHAR(20) | 气象站ID |
| station_name | VARCHAR(200) | 气象站名称 |
| year | INT | 年份 |
| obs_month | INT | 月份(1-12) |
| latitude | DOUBLE | 纬度 |
| longitude | DOUBLE | 经度 |
| avg_temp | DOUBLE | 月均温(°C) |
| avg_temp_max | DOUBLE | 月均最高温(°C) |
| avg_temp_min | DOUBLE | 月均最低温(°C) |
| max_temp | DOUBLE | 月最高温(°C) |
| min_temp | DOUBLE | 月最低温(°C) |
| total_precip | DOUBLE | 月总降水量(mm) |
| rainy_days | INT | 降雨天数 |
| extreme_days | INT | 极端天气天数 |
| heat_wave_days | INT | 热浪天数 |
| cold_wave_days | INT | 寒潮天数 |
| frost_days | INT | 霜冻天数 |
| snow_days | INT | 降雪天数 |
| thunder_days | INT | 雷暴天数 |
| obs_days | INT | 观测天数 |
| climate_zone | VARCHAR(20) | 气候带 |

主键: `(station_id, year, obs_month)`

**气候带分类:**
| 值 | 中文 | 判定条件 |
|----|------|----------|
| tropical | 热带 | 月均温≥18°C 且 月最低温≥18°C |
| temperate | 温带 | 月均温≥10°C |
| arid | 干旱 | 月均降水<10mm/天 |
| polar | 寒带 | \|纬度\|≥60° 或 月均温<0°C |
| continental | 大陆性 | 其他 |

## ADS 层 (MySQL)

### ads_kpi — KPI看板

| 字段 | 类型 | 说明 |
|------|------|------|
| kpi_name | VARCHAR(50) | 指标名称 |
| kpi_value | DOUBLE | 指标值 |
| kpi_unit | VARCHAR(20) | 单位 |
| kpi_desc | VARCHAR(200) | 中文描述 |
| data_year | INT | 数据年份 |

主键: `(kpi_name, data_year)`

**预置指标:**
| kpi_name | 说明 |
|----------|------|
| global_avg_temp | 全球年平均温度 |
| total_stations | 活跃气象站总数 |
| extreme_event_pct | 极端天气日数占比 |
| hottest_station_temp | 年度最高气温 |

---

### ads_ranking — 站点排名

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 自增主键 |
| category | VARCHAR(20) | 排名类别 |
| rank_num | INT | 排名 |
| station_id | VARCHAR(20) | 气象站ID |
| station_name | VARCHAR(200) | 气象站名称 |
| value | DOUBLE | 排名值 |
| data_year | INT | 数据年份 |

唯一键: `(category, data_year, rank_num)`

**类别:**
| category | 说明 | 单位 |
|----------|------|------|
| hottest | 最热 | °C |
| coldest | 最冷 | °C |
| rainiest | 降水最多 | mm |
| most_extreme | 极端天气最多 | 天 |

---

### ads_monthly_trend — 月度趋势

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| obs_month | INT | 月份 |
| avg_temp | DOUBLE | 全球月均温(°C) |
| avg_max | DOUBLE | 全球月均最高温(°C) |
| avg_min | DOUBLE | 全球月均最低温(°C) |

主键: `(data_year, obs_month)`

---

### ads_zones — 气候带分布

| 字段 | 类型 | 说明 |
|------|------|------|
| data_year | INT | 年份 |
| climate_zone | VARCHAR(20) | 气候带 |
| station_count | INT | 站点数量 |

主键: `(data_year, climate_zone)`
